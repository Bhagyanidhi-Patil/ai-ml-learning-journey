ML is process of teaching computers to learn patterns from data and make decisions based on those patterns.

Training
Training is when the model learns. Teaching computer to learn patterns from data is training. 

Inference
Inference is when the trained model is used. Make decision based on those patterns are inference.

    Goal: There is a bank and you have to design a system which decides whether a customer should get a loan or not.

ML types :

1. Supervised Learning
    Supervised Learning = learning with answers
    Data has input + correct output (labels). The model learns by example.
    Examples:
        ‚Ä¢ Loan approval (approve / reject)
        ‚Ä¢ Email spam detection
        ‚Ä¢ House price prediction

    Type 1: Classification
        üëâ Output is a word / category. Output can be binary (like yes or no) or multi-class also. 
        You are asking:
            ‚ÄúWhich group does this belong to?‚Äù
        Examples:
            ‚Ä¢ Email ‚Üí Spam or Not Spam
            ‚Ä¢ Loan ‚Üí Yes or No
            ‚Ä¢ Photo ‚Üí Cat or Dog
        üîë Output choices are limited and fixed.

    Type 2: Regression
        üëâ Output is a number
        You are asking:
            ‚ÄúHow much?‚Äù or ‚ÄúHow many?‚Äù
        Examples:  Swiggy delivery time prediction is a perfect example of supervised learning (regression)
            ‚Ä¢ House price ‚Üí ‚Çπ45,00,000
            ‚Ä¢ Temperature ‚Üí 32.5¬∞C
            ‚Ä¢ Salary ‚Üí ‚Çπ80,000
        üîë Output is a number that can change continuously.

2. Unsupervised Learning
    Data has only input, no output labels. 
    In unsupervised learning, input data is given, but it is not labeled or neatly categorized. 
    The ML algorithm itself finds patterns and separates the data into groups (clusters) and anomaly / outliers (Outliers are data points
    that are very different from the rest of the data - rare events, errors) .The model finds hidden patterns on its own. 
    
    What the model does
    It tries to find structure on its own, mainly by:
        ‚Ä¢ grouping similar things together (clusters)
        ‚Ä¢ finding hidden patterns

    Type 1 Clustering:
        A clustering problem in unsupervised learning is when we want to group similar data points together without having any labels.

        What makes it a ‚Äúproblem‚Äù
            ‚Ä¢ You have only input data
            ‚Ä¢ You don‚Äôt know the correct groups
            ‚Ä¢ The algorithm must discover the groups by itself
        Easy example
        Imagine you have customers with data like:
            ‚Ä¢ age
            ‚Ä¢ income
            ‚Ä¢ shopping behavior
        No labels like ‚Äúpremium‚Äù or ‚Äúbudget‚Äù.
        Clustering will group them into:
            ‚Ä¢ similar spending habits
            ‚Ä¢ similar age/income patterns
        Later, humans may name the clusters.
    
    Type 2 Associating:
        The association problem (or association rule learning) in unsupervised learning is about finding relationships between items in data.
        Association problem = discovering which things often occur together.

        Example
        Amazon / Blink it suggest items together because association algorithms learn which products are often bought together.
        So:
            ‚Ä¢ Many customers buy milk
            ‚Ä¢ A lot of those same customers also buy bread
            ‚Ä¢ The system learns this pattern
            ‚Ä¢ When you add milk ‚Üí it suggests bread

        Association learning finds ‚Äúitems that go together‚Äù based on past behavior.

3. Reinforcement Learning
    Reinforcement Learning is learning by trial and error using rewards and penalties. There are no correct answers given upfront.
    Reinforcement learning teaches an agent to make decisions by rewarding good actions and penalizing bad ones.
    
    How it works (step by step)
        1. Agent takes an action
        2. Environment responds
        3. Agent gets a reward (good) or penalty (bad)
        4. Agent learns which actions give more reward
    Repeat this many times ‚Üí behavior improves.

    Simple example (real life)
        Teaching a dog üê∂:
            ‚Ä¢ Sit ‚Üí treat (reward)
            ‚Ä¢ Jump on sofa ‚Üí no treat (penalty)
        Over time, the dog learns what to do.

        At first:
            ‚Ä¢ Dog tries random actions
            ‚Ä¢ Sometimes gets treat, sometimes not
        After many repeats:
            ‚Ä¢ Dog notices:
        ‚ÄúWhen I sit, I get food‚Äù
            ‚Ä¢ So it chooses sitting more often
        No one tells the dog:
            ‚Ä¢ ‚ÄúSit is correct‚Äù
            ‚Ä¢ ‚ÄúJump is wrong‚Äù
        The dog figures it out from rewards.
