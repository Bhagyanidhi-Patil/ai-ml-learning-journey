What is Deep learning? 

Deep learning is a type of machine learning that uses many layers of artificial neural networks to learn very complex patterns from data.
A neural network is a way to teach computers by loosely copying how the human brain works.
A neural network is a system of layered neurons that learns patterns from data by adjusting connections.

How it learns (high level)
	1. Guess an answer
	2. Check how wrong it is
	3. Adjust connections slightly
	4. Repeat many times
Over time, guesses get better.
